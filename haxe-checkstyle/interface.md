toc: false
---

Checks and enforces interface style. Either to allow properties and methods or just methods. Has an option to `allowMarkerInterfaces`.

### Configuration

```json
{
    "type": "Interface",
    "props": {
        "allowProperties": false,
        "allowMarkerInterfaces": true,
        "severity": "INFO"
    }
}
```
