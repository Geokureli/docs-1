Spacing check on if, for, while, switch, try statements and around operators.

### Configuration

```json
{
    "type": "Spacing",
    "props": {
        "severity": "INFO",
        "spaceIfCondition": true,
        "spaceAroundBinop": true,
        "noSpaceAroundUnop": true,
        "ignoreRangeOperator": true
    }
}
```
