Spacing check on if statement and around operators.

### Configuration

```json
{
    "type": "Spacing",
    "props": {
        "severity": "INFO",
        "spaceIfCondition": true,
        "spaceAroundBinop": true,
        "noSpaceAroundUnop": true,
        "ignoreRangeOperator": true
    }
}
```
