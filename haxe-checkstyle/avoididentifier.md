Checks for identifiers to avoid. (available since 2.6.0)

### Configuration

```json
{
    "type": "AvoidIdentifier",
    "props": {
        "severity": "INFO",
        "avoidIdentifiers": []
    }
}
```

**`avoidIdentifiers`** - List of identifiers that should no longer be used in your code.
