A check for `TODO/FIXME/HACK/XXX/BUG` comments. The format can be customised bu changing `format` property.

### Configuration

```json
{
    "type": "TODOComment",
    "props": {
        "format": "TODO|FIXME|HACK|XXX|BUG",
        "severity": "IGNORE"
    }
}
```
